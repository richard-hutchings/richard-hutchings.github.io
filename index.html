Hi there

I am a self taught coder and have been studying Computer Science and Programming since early 2018, when I decided to change careers.
I am owner/ operator of a small Landscape/ Gardening business for the last 11 years. However a few years ago I suffered a shoulder injury which led me to question the longterm nature of my work.
It ultimately saw me embark upon a path of the discovery of the wonders of programming and the creation of Web Apps, Websites and Gaming.
I was amazed by how things work on the inside and the different elements that pull things together.

At present I am building websites for friends and family, Coding games in Unity with C#, Creating CSS and React Animations.
I particulary like working with Ruby, React, JavaScript, Bootstrap, CSS, C# and Unity.
I have created apps with Test Driven Deployment, MVC(models, views, controllers), Database migrations, SQL,
I am familiar using AWS Cloud9, Visual Studio Code, command line interface, Git/Github for version Control and have also coded projects using JQuery, NodeJS, PHP, C.
